# koishi-plugin-baidu

[![npm](https://img.shields.io/npm/v/koishi-plugin-baidu?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-baidu)

在 Koishi 中搜索 [百度百科](https://baike.baidu.com)。

## 配置项

### maxResultCount

- 类型: `number`
- 默认值: `3`

最多返回的结果数量。

### maxSummaryLength

- 类型: `number`
- 默认值: `200`

最大返回的摘要长度。
